# Proyecto final Streamlit: Fotogrametría UAV

Aplicación web interactiva desarrollada con Streamlit a partir de la Tarea 3 de Fotogrametría.

App publicada: https://proyectofinal-fuz7hdjmgbaxzwatpxsumb.streamlit.app/

## Archivos incluidos

- `app.py`: código principal de la aplicación.
- `requirements.txt`: bibliotecas necesarias para ejecutar la aplicación.
- `datos.csv`: base de datos usada en la Tarea 3.

## Ejecución local

```bash
pip install -r requirements.txt
streamlit run app.py
```
